# 行情

**行情 (Quote)** 是指市場某一特定時點的資產價格相關資訊，例如買價和賣價等。

Eskmo 中行情分為 **即時行情** 與 **歷史行情**，兩者操作方式不同。

並且在不同商品別（證券、期貨等）之下，提供的行情資訊也略有不同。
